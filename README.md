# visionOS Portfolio

A collection of visionOS projects exploring spatial computing on Apple Vision Pro.

## Projects

| Project | Description |
|---------|-------------|
| [MetalCube](MetalCube/) | A volumetric window app using Metal shaders and RealityKit to render a cube in 3D space. |

## Requirements

- Xcode 16+
- visionOS 2.0+ SDK
- Apple Vision Pro simulator or device

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/visionOS-portfolio.git
   ```
2. Open any project's `.xcodeproj` in Xcode.
3. Select the visionOS simulator or device as the run destination.
4. Build and run.
